Into this:

For the roleplaying game enthusiasts, a new way to map out your campaigns and encounters.

Thisa work in progress is currently able to create a map where the user sets the size, regions within the map where the user sets their location and size, paths from region to region, and the regions are currently able to contain objects, npcs and enemies.

If you would like to use this language

1. Go to the GitHub link on the top right of this webpage and download the MAPR.py and RPGMAPR.tx files(test.map is optional if you would like a baseline for a program).

2. Make sure you have python 3.12 or greater installed and that you have both textX and matplot installed.

3. Create your .map program and run the MAPR.py parser, naming your .map file in the terminal to display your map and print the descriptions.
